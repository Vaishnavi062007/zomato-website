Project Overview:
This project replicates the Zomato landing page design, including:
Hero section with background image & search bar
Navigation bar with links
Food categories section
Collections with offers & ratings
Popular localities around OMR (Chennai)

Footer section:
The goal of this project is to strengthen frontend fundamentals and understand real-world website structure.

Technologies Used:

HTML5 – Page structure
CSS3 – Styling & layout
Font Awesome – Icons
Google Fonts / System Fonts
Online images – For UI visuals

 Features:

Responsive layout using Flexbox & Grid
Modern UI inspired by Zomato
Hover effects on cards
Clean and structured code

Beginner-friendly project

Project Structure:

How to Run the Project:

Download or clone this repository
